# Screenshots

Drop the following images here so the main README renders cleanly. PNG, ideally width 1200 to 1600 px.

| Filename | What to capture |
|---|---|
| `ui-hero.png` | Full UI with a call in progress, transcript populated. Wide hero shot for the top of the README. |
| `ui-idle.png` | UI before placing a call. Number field, scenario selector, Start button. |
| `ui-in-call.png` | UI mid-call, transcript streaming, status dot green. |
| `call-logs.png` | Twilio Console call logs showing successful completed outbound calls. Crops out the side nav. |

Quick capture recipe (Windows):

1. Run the app and ngrok, place a call to your own number.
2. While the call is live, press `Win + Shift + S`, drag over the browser window, paste into Paint, save as PNG.
3. For the Twilio console, log in, navigate to Voice → Calls, screenshot the table.

If a file is missing, the README image tag renders as a small broken-image icon, which is harmless but does look unfinished. Add the four images above and the README is photo-ready.
